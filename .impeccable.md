# Impeccable Design Context — Nami Garden

## Project
Nami Garden is a private couple-finance product for Amir + Ayunni. It turns household money into a calm garden ritual: fast logging, gentle status, and a monthly money date. The emotional goal is clarity without blame.

## Design direction
Warm fintech, not childish game UI. Use creature language lightly, as emotional labels for household finance categories. The product should feel adult enough for real money, soft enough for couple conversations, and fast enough for daily use.

## Visual references
- Wise: clear fintech confidence, lime accent, rounded cards, legible money UI.
- Lovable: warm cream background, human tone, soft borders, editorial calm.
- Figma: playful restraint and confident product surfaces, but avoid over-bright rainbow UI.

## Design system
- Background: warm cream `#F8F3E8`
- Primary text: garden charcoal `#172018`
- Muted text: moss gray `#687365`
- Primary accent: fresh lime `#B8F36B`
- Secondary accent: soft mint `#DFF7C7`
- Warm surface: `#FFFDF7`
- Border: `rgba(23,32,24,0.10)`
- Amber attention: `#F4C96B`
- Soft coral only for gentle attention: `#F5A38B`

## Typography
- Use `DM Sans` for warm friendliness.
- Use tight but readable headings.
- Avoid generic Inter-everywhere feel unless building production system later.

## Component rules
- Large mobile-first tap targets.
- Pill buttons for primary actions.
- Cards should use soft borders, not heavy shadows.
- One primary action per screen.
- Creature cards should be simple status objects, not loud cartoons.
- Money amounts must be highly legible.

## Tone rules
Use:
- “steady”
- “heads-up”
- “check in”
- “needs care”
- “worth discussing”

Avoid:
- “failed”
- “bad”
- “debt”
- “you owe”
- “boss battle”
- red-alert language

## Anti-patterns to avoid
- No purple Tailwind gradients.
- No generic SaaS hero → features → CTA layout.
- No cards inside cards.
- No thick left-accent side-tab cards.
- No dark fintech dashboard that feels like trading software.
- No childish Pokémon clone UI.
- No scoreboards comparing Amir vs Ayunni.
- No shame-coded red states for personal spending.

## MVP screens to design
1. Garden Home
2. Fast Add Entry
3. Monthly Nami Review

## Quality bar
The design should pass this sentence:
“Ayunni would not feel monitored, and Amir would still trust the numbers.”
